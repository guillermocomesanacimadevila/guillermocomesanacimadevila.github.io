Featured Projects

MR-CoPe: Automated Mendelian Randomisation Engine

A comprehensive toolkit for automated Mendelian Randomisation analysis, designed to streamline causal inference in genetic epidemiology.

PA-Predict: Machine Learning Pipeline to Diagnose Pernicious Anaemia

A robust machine learning pipeline to support the diagnosis of Pernicious Anaemia from clinical and laboratory data.

ENIGMA-Pipeline: Working Memory Prediction from sMRI Data

Machine learning for predicting working memory outcomes from structural MRI data in schizophrenia research. (Internal academic project)

CELLFLOW: Deep Learning for Live-cell Microscopy

Self-supervised representation learning and U-Net classifier for identifying apoptosis and mitosis from spatiotemporal live-cell microscopy.
